小鎮風情

鹿港鎮，在彰化，臺灣史上名氣大。 
先民冒險過黑水，聚落成鎮傳佳話。

龍山寺，天后宮，雕刻精美唐山工。
敬神禮佛遊客多，拜完媽祖拜關公。

九曲巷，走迷宮，方磚古道滿地紅。
十宜樓上看風景，品茶望月對清風。

牛舌餅，鳳眼糕，大街小巷香味飄。
如果想吃古早味，請來鹿港走一遭。

回到鹿港

我們的老家在鹿港，那裡的建築古色古香。每次回老家，我們都會到處逛一逛。  

冬天，鹿港的風很大，九曲巷彎彎曲曲的，風一點也吹不進來，很溫暖。走在巷子裡，好像在玩捉迷藏一樣，一不留神，爸爸、媽媽就不見了。爸爸說，鹿港從前是一個商港，常有海盜出沒。媽媽也告訴我們，九曲巷除了防風以外，還可以防盜。你看！彎曲的巷子好像迷宮一樣，聽說不少海盜還真的曾經在裡面「迷路」呢！ 

逛完了九曲巷，我們再去參觀那口著名的「半邊井」。紅磚砌成的半邊井，一半在牆內，一半在牆外，好奇怪呵！奶奶說，由於鹿港人很有人情味，所以有些人把井挖在牆腳下，讓沒錢挖井的人，也能來打水喝。奶奶還說，她就是喝這一口井水長大的，我聽了很感動。 

我喜歡回鹿港，因為那個地方，不但好玩，還很有人情味。
